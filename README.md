# Ledge Link
Ledge Link Android SDK.

For more information, see the [website](https://www.ledge.me/).

## Download

Maven:

```xml
<dependency>
  <groupId>us.ledge.link</groupId>
  <artifactId>sdk</artifactId>
  <version>1.0</version>
</dependency>
```

Or Gradle:

```groovy
compile 'us.ledge.link:sdk:1.0'
```

## Requirements

* Java version: 7+
* Android version: 4.0.3+
